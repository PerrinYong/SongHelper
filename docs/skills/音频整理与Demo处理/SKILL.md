# 技能：音频整理与 Demo 处理

## 这个技能解决什么问题

当你只是想快速把素材整理成一个可听、可发、可比对的 demo 版本时，用这个技能。

## 典型需求

- 拼接几个片段
- 给人声或整段音频加一点空间感
- 统一多个版本的响度

## 常用底层工具

- `concat-audio`
- `apply-reverb`
- `normalize-audio`

## 输入

- 一段或多段音频

## 输出

- 拼接版音频
- 混响版音频
- 标准化版音频

## 适用场景

- 做 rough mix
- 准备试听版
- 为后续创作迭代快速导出版本

## 注意事项

- 这类处理适合 demo 级整理，不等于专业混音 / 母带
