# 技能：创作策划与风格拆解

## 这个技能解决什么问题

当你的问题还不在“处理音频”，而是在“这首歌该怎么成立”，这个技能更有用。

## 典型需求

- 主题怎么立？
- 主歌/副歌如何分工？
- 这首歌为什么不像你、为什么没有记忆点？
- 应该借鉴参考曲的什么，不该借什么？
- MV、视觉、提示词怎么和音乐统一？

## 常用底层支撑

- 歌曲分析结论
- 旋律转写结果
- 参考曲比对结果
- 人工风格判断与文本推理

## 输入

- 文字描述、歌词、参考曲、音频版本、角色设定、视觉方向

## 输出

- 创作方向建议
- 结构建议
- 歌词 / hook 建议
- 风格拆解结论
- 提示词与视觉方向建议

## 适用场景

- 新歌立项
- AI 生成结果诊断
- 改词、改旋律、改结构
- 视觉联动规划

## 注意事项

- 这是高层创作协作能力，不一定对应单个 CLI 命令，但会使用底层分析结果作为依据
