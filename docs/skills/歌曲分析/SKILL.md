# 技能：歌曲分析

## 这个技能解决什么问题

当你拿到一首现有歌曲、demo、录音或 AI 生成结果，想先看清它“整体长什么样”，这个技能最合适。

## 典型需求

- 这首歌大概多少 BPM？
- 是什么调？偏大调还是小调？
- 结构上大概怎么分段？
- 哪些地方像主歌，哪些地方像副歌？

## 常用底层工具

- `probe-audio`
- `analyze`

## 输入

- 单首音频文件

## 输出

- 音频探测 JSON
- 分析 JSON
- 文字解释结论

## 适用场景

- 学参考曲
- 分析自己当前版本
- 判断后续是否需要转写或重编

## 注意事项

- BPM / 调性 / 段落是“分析辅助结论”，需要耳朵最终确认
