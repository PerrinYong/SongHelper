# 技能：主旋律转写

## 这个技能解决什么问题

当你已经有 vocal、哼唱、单旋律录音，想把它变成更可编辑、更可分析的旋律数据时，用这个技能。

## 典型需求

- 从人声提主旋律
- 导出 MIDI
- 转成简谱草稿
- 清理掉碎音和异常高音

## 常用底层工具

- `transcribe`
- `clean-melody`
- `midi-to-jianpu`
- `human-jianpu`

## 输入

- 单轨旋律音频，或相对干净的 vocal stem

## 输出

- MIDI
- 音符 JSON
- 简谱草稿
- 清理后主旋律版本

## 适用场景

- 做主旋律归档
- 给编曲或乐手看草稿
- 比较多个旋律版本

## 注意事项

- 当前结果更适合作为“草稿与分析底稿”，不是正式出版谱
