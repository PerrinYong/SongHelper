# 歌曲创作辅助工作流

## A. 灵感与定位

输入：主题、情绪、参考曲、目标受众

输出：

- 风格关键词
- 歌曲结构草图
- 目标 BPM / 调性候选
- 参考曲拆解结果

## B. Demo 分析

输入：手机录音、钢琴弹唱、已有伴奏、小样

输出：

- 节拍 / 调性估计
- 段落切分
- 旋律骨架
- 和弦候选

## C. 声部拆分与整理

输入：混合音频

输出：

- vocals
- instrumental
- drums / bass / other（如可用）
- 清理后的片段文件

## D. 旋律转写

输入：vocal stem 或单旋律音频

输出：

- MIDI
- 音高时间序列
- 简谱 / 音名草稿
- 乐谱草稿

## E. 编曲与制作辅助

输入：旋律、和弦、参考曲方向

输出：

- 配器建议
- 层次推进建议
- 节奏型建议
- 和声 / backing vocal 建议

## F. 混音与审听

输入：rough mix / stems / 参考曲

输出：

- 问题清单
- 效果链建议
- 空间与频段平衡建议
- revision plan

## G. 导出与归档

输入：最终版本与过程文件

输出：

- exports
- analysis reports
- score drafts
- version notes
