# SongHelper 能力清单

## 1. 创作前期 / 方向定义

- 主题拆解：从一句话扩展为情绪、叙事视角、关键词簇
- 风格定位：拆解参考曲的时代感、节奏密度、编曲层次、vocal 气质
- 结构设计：主歌/预副歌/副歌/桥段/尾奏安排
- 歌词辅助：押韵、节奏口语化、意象一致性、hook 句强化

## 2. 音乐分析

- BPM / 节拍估计
- 调性与转调判断
- 和弦候选与进行分析
- 旋律轮廓与动机复现分析
- 段落切分、能量分布、高潮位置分析
- 参考曲对比：速度、调性、情绪、动态、频段占比

## 3. 音频处理

- vocal / accompaniment / drums / bass / other 分离
- 录音裁剪、拼接、静音段清理
- 响度归一、峰值检查、基础降噪建议
- 效果链建议：EQ / compression / reverb / delay / saturation
- 版本导出：demo、伴奏、acapella、练唱版

## 4. 旋律与乐谱

- 从人声提取音高轨迹
- 从旋律生成 MIDI
- 输出简谱 / 音名序列 / 节奏草稿
- 生成练习用 lead sheet 草案
- 比较不同旋律版本的跳进级进比例与可唱性

## 5. 编曲与制作辅助

- 编曲层次建议：intro / verse / chorus 的乐器进出
- 节奏型参考：鼓组律动、贝斯走向、铺底密度
- 和声 / 复调 / 叠唱建议
- 配器替代建议：某乐器太满时给出减法方案

## 6. 混音与审听辅助

- 基于文字给出混响、delay、声像、频段拥挤建议
- 对比参考曲给出混音方向差异
- 识别常见问题：齿音、低频浑浊、主唱被埋、空间不统一
- 输出 revision checklist

## 7. 项目管理与归档

- 作品版本命名规范
- 创作日志、修改记录、待办事项整理
- 导出物归档：stems、scores、analysis、mixes、exports
- 为单首歌建立标准工作目录

## 8. 我可通过哪些手段帮你

### 纯文本 / 推理

- 创作 brainstorming
- 乐理分析解释
- 方案比较与决策建议
- 制作问题排查

### 程序编写

- 编写音频分析脚本
- 编写批处理流水线
- 编写导出与整理工具
- 编写报告生成器

### 工具调用 / 本地工程

- 在工作空间组织素材与输出
- 调用 Python 脚本处理音频
- 调用第三方 CLI 工具（安装后）执行分离、转换、导出

### LLM Agent / Skill

- 做复杂流程设计与拆解
- 生成可执行脚本骨架
- 帮你将创作过程沉淀为标准化工作流

## 9. 你后续可能还会用到的功能

- 参考曲 A/B 对比报告
- 歌词与旋律重音对齐检查
- 人声双轨 / 和声编排建议
- 副歌 hook 密度与记忆点分析
- 自动生成练唱版（降速 / 升降调）
- 多版本旋律差异对照
- 一键导出“投稿包 / 练习包 / 存档包”
