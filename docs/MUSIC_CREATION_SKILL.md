# 音乐创作协作 Skill（合并版）

该文档从 `music_agent_workspace/AGENT_SKILL_MUSIC_CREATION.md` 提炼并并入当前工程，作为 SongHelper 的音乐任务执行约束。

## 目标

在歌曲创作过程中，同时提供：

- 文本层面的创作协助
- 程序化的分析与处理能力
- 对结果置信度与边界的明确说明

## 任务分类

### 1. 创作设计类

- 主题
- 歌词
- 结构
- 风格
- 提示词
- MV / 分镜

### 2. 分析类

- 节奏
- 调性
- 旋律
- 演唱方式
- 版本差异

### 3. 处理类

- 分离
- 提取
- 拼接
- 混响
- 标准化

### 4. 自动化类

- 批量分析
- 批量导出 MIDI / CSV / JSON
- 批量生成提示词模板

## 输出原则

### 必须区分置信度

- 稳定结论
- 粗估结论
- 需要人工回听确认的结论

### 不夸大工具能力

- HPSS / 中置提取 ≠ 高质量 AI stem 分离
- `pyin` 单旋律估计 ≠ 出版级最终乐谱

## 遇到音频时的推荐顺序

1. 探测文件信息
2. 判断是否适合直接旋律提取
3. 必要时先做粗分离
4. 再做旋律提取
5. 输出 CSV / MIDI / 摘要
6. 再由 LLM 做音乐解释

## 可复用输出模板

### 分析结论

- 文件信息
- 节奏估计
- 调性估计
- 音域
- 主旋律概览
- 置信度说明
- 后续建议

### 处理结果

- 输入文件
- 执行命令 / 脚本
- 输出文件
- 适用场景
- 局限
