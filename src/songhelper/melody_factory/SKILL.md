# 技能：主旋律发明工作流（技术链路）

## 这个技能当前解决什么问题

它解决的不是“让 AI 直接写歌”，而是把一个已经确定好的主旋律结构表示落成可处理的技术资产。

当前能力包括：

- 维护 MelodySpec 结构
- 导出 ABC
- 导出 MIDI
- 导出可试听音频

## 当前适用场景

- 你已经人工或借助 Agent 确定了一条 4 小节 hook 草稿
- 你需要快速把它转成可试听、可筛选、可导出的版本

## 当前输入

- MelodySpec JSON
- 或 ABC 文本

## 当前输出

- ABC
- MIDI
- WAV

## 当前边界

- 暂不负责灵感生成
- 暂不负责 MusicXML
- 暂不负责浏览器试听 UI
